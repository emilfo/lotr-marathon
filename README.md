# Middle-earth Marathon Site

Static multi-page GitHub Pages site for a 1-3 May LOTR + Hobbit marathon.

## Pages

- `index.html` - epic landing page + countdown
- `itinerary.html` - detailed schedule with correct movie names
- `movies.html` - all six movies with trailer links and movie-order easter egg
- `music.html` - soundtrack links + quote generator
- `doors-of-durin.html` - hidden easter egg page unlocked by typing `mellon`

## Easter eggs

- Click the ring button 9+ times
- Enter Konami code
- Type `mellon`
- Click all movie titles in exact order on `movies.html`

## Run locally

Open `index.html` in your browser.

## Publish on GitHub Pages

1. Create a repo on GitHub and push this folder.
2. In repo settings, enable Pages from the `main` branch root.
3. Wait for deploy and open your live URL.

## Media note

This site links to official trailer/soundtrack pages rather than bundling copyrighted movie assets directly.
